Project Overview

This project is an ATM application where users can perform basic banking transactions such as checking balances and withdrawing money. The application provides an interactive user interface and ensures secure transactions. It was developed using Node.js, Express, SQL, and C++ for backend processing and functionality.

The aim of the project was to build a system that simulates the operations of an ATM. The architecture of the target system follows a client-server model. In this project, the client is the ATM machine, which consists of various system components that were defined in detail during the project specification phase. The server machine contains a database server and other software that were also defined during the project specification phase. Data sent from the client application is stored in a database on the server machine via the network.
