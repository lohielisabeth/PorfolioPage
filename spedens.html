Project Overview

The Arduino Speed Game project involved designing and implementing a reaction-based game using Arduino. The game utilized 5 buttons for player input, LEDs for visual feedback, and a 7-segment display to show scores. Buttons were connected with debouncing handled programmatically, and interrupts were used to detect presses. A Timer1 interrupt generated random sequences of numbers, which players had to replicate using buttons. Correct inputs increased the game's speed, while incorrect inputs ended the game. The display module used two serial-to-parallel shift registers (74HC595) to drive the 7-segment displays without relying on built-in Arduino libraries. LEDs and buttons were abstracted through modular functions, ensuring flexibility and clear game logic. The game logic managed sequences, compared user input against generated patterns, and displayed results dynamically. Timer-based difficulty adjustments ensured consistent challenge levels across implementations.
