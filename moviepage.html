About the Movie Page Project

The project involved developing a dynamic and responsive web application for movie enthusiasts. The frontend was built using React, and the backend was powered by Node.js. PostgreSQL was used as the database for storing user data and other relevant information. The application integrates with open data sources, including The Movie Database (TMDb) API for movie information and the Finnkino API for showtimes. Users can search for movies, view Finnkino showtimes, and leave reviews. By signing in, users can add movies to their favorites and provide feedback on watched movies. Node.js and SQL were utilized to manage movie data and ensure secure user authentication.
